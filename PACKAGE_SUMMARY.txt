ğŸ”§ WHAT THE INSTALLER DOES
---------------------------
Step 1: Check prerequisites (OpenClaw, npm, Node.js)
Step 2: Install dependencies (mcporter, notebooklm-mcp via npm)
Step 3: Authenticate NotebookLM (opens Chrome, saves tokens)
Step 4: Create mcporter config (~/.openclaw/mcporter.json)
Step 5: Create skill (SKILL.md + notebooklm.sh wrapper)
Step 6: Update OpenClaw config (adds skill entry)
Step 7: Restart OpenClaw daemon
Step 8: Verify installation (test skill wrapper)
